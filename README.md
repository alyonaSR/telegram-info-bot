# Telegram Info Bot

Pet-проект на Python: Telegram-бот, который умеет отвечать на сообщения пользователя, отправлять случайные цитаты, показывать погоду по городу и собирать заголовки новостей из нескольких источников.

## Описание проекта

Проект реализует Telegram-бота через Telegram Bot API без использования готовых bot-фреймворков.  
В проекте есть две версии бота:

- синхронная версия на `requests`;
- асинхронная версия на `aiohttp` и `asyncio`.

Основная версия проекта — асинхронная. Она поддерживает обработку команд, работу с внешними API, long polling, простое состояние пользователя и конкурентный сбор данных из нескольких источников.

## Функциональность

Бот поддерживает команды:

- `/start` — приветственное сообщение и список доступных команд;
- `/quote` — получение случайной цитаты;
- `/weather` — запрос города и вывод текущей погоды;
- `/headlines` — получение заголовков новостей из нескольких источников;
- любое другое сообщение — бот повторяет текст пользователя.

## Что реализовано

- работа с Telegram Bot API;
- получение сообщений через long polling;
- отправка сообщений пользователю;
- обработка команд;
- простая FSM-логика для ожидания ввода города;
- получение погоды через OpenWeatherMap API;
- веб-скрапинг цитат с сайта quotes.toscrape.com;
- веб-скрапинг новостных заголовков;
- асинхронные HTTP-запросы через `aiohttp`;
- конкурентный сбор новостей через `asyncio.gather`;
- обработка ошибок при недоступности внешних сервисов;
- хранение токенов и API-ключей в `.env`.

## Используемые технологии

- Python
- asyncio
- aiohttp
- requests
- BeautifulSoup4
- python-dotenv
- Telegram Bot API
- OpenWeatherMap API
- HTML parsing / web scraping
